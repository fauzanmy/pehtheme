# Pre-11ty Template Archive

This folder contains the legacy static HTML snapshot used before Pehtheme migrated to the Eleventy + Nunjucks + Tailwind CSS workflow.

## Purpose

This archive is kept for historical reference only. It is not part of the active build system.

## Notes

- Active source files now live in `/src`
- Generated production output now lives in `/dist`
- This archived snapshot is preserved only as a reference for the earlier template structure
